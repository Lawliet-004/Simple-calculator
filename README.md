# 🧮 Simple calculator

Hi there! I'm a beginner programmer currently learning C#, and this is one of my projects built with it!

## Features

- Addition and subtraction of two integers
- Input validation to ensure only valid numbers are accepted
- Simple and clean console interface

## How to Run

> **Requirements:** [.NET SDK](https://dotnet.microsoft.com/download) installed on your machine.

1. **Download or clone the repo**
   ```bash
   git clone https://github.com/Lawliet-004/Simple-calculator.git
   ```

2. **Open a terminal in the project folder**
   ```bash
   cd Simple-calculator
   ```

3. **Run the app**
   ```bash
   dotnet run
   ```
